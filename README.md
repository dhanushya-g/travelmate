---

## 📖 Overview

TravelMate is a smart trip planning platform that helps users organize their journeys with ease. It provides an intuitive interface for discovering destinations, planning itineraries, managing travel information, and enhancing the overall travel experience.

The application is designed with a responsive UI and integrates Supabase for backend services such as authentication and data management.

---

## ✨ Features

- 🌍 Search and explore destinations
- 📅 Smart trip planning
- 🔐 Secure user authentication
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🎨 Modern UI using Tailwind CSS & shadcn/ui
- ☁️ Supabase backend integration
- 💾 Real-time database support

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/dhanushya-g/travelmate.git
```

### Navigate to the Project

```bash
cd travelmate
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
